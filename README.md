### 🚀 Sobre mí

Soy un desarrollador de software de **21 años** y estudiante universitario enfocado en la intersección entre el **hardware y el software**. Me apasiona crear soluciones que impulsen la **sostenibilidad (Cleantech)** y la eficiencia a través de la tecnología.

* 🛠️ **Enfoque actual:** Desarrollo de sistemas **IoT** para la gestión inteligente de energía y monitoreo agrícola.
* 🔋 **Proyecto Destacado:** **FaradIA**, una plataforma de monitoreo energético diseñada para optimizar el consumo mediante análisis de datos en tiempo real.
* 🎨 **Filosofía de diseño:** Fiel creyente del minimalismo, las interfaces modernas y el código bien estructurado.
* 🎮 **Intereses:** Me fascina la arquitectura técnica de servidores, la lectura de historias sobre evolución y estrategia, y el diseño de interfaces UI/UX de alto impacto.
